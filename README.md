# Scientific Research & Conference Presentations

Acest repository reunește proiectele de cercetare academică și prezentările susținute în cadrul sesiunilor de comunicări științifice studențești. Temele abordate reflectă interesul meu pentru matematica aplicată, modelarea sistemelor complexe și cercetarea multidisciplinară.

## 🔬 Proiecte de Cercetare

### 1. Sisteme haotice descrise cu ajutorul calculului fracționar
- **Tema:** Analiza sistemelor haotice din perspectiva calculului fracționar, cu aplicații în modelarea fenomenelor complexe (ex: simularea pandemiei COVID-19 prin modelul SEIR fracționar).
- **Resurse:** [Prezentare Proiect (PPTX)](Sisteme-haotice-calcul-fractionar-Presentation.pptx)
- **Competențe:** Teoria haosului, calcul fracționar, modelare numerică.

### 2. Aplicații ale funcțiilor meromorfe în teoria circuitelor
- **Tema:** Studiul circuitelor RLC și a stabilității sistemelor dinamice folosind analiza complexă și punctele singulare (poli și zerouri).
- **Resurse:** - [Raport Tehnic (PDF)](Meromorphic-Functions-Applications-Report.pdf)
  - [Prezentare Proiect (PPTX)](Meromorphic-Functions-Applications-Presentation.pptx)
- **Competențe:** Analiză complexă, circuite electrice, stabilitatea sistemelor.

### 3. Impactul stresului asupra succesului academic
- **Tema:** Cercetare cantitativă privind influența factorilor emoționali asupra performanței studenților, bazată pe anchetă sociologică.
- **Resurse:**
  - [Raport de Cercetare (PDF)](Impact-of-Stress-Academic-Success-Report.pdf)
  - [Prezentare Rezultate (PPTX)](Impact-of-Stress-Academic-Success-Presentation.pptx)
- **Competențe:** Metodologia cercetării, analiză statistică, colectare și interpretare de date.

## 🎓 Participări și Validare
Fiecare proiect a fost susținut public în cadrul Sesiunilor de Comunicări Științifice Studențești, fiind evaluat de comisii de specialitate. Aceste lucrări demonstrează abilitatea de a sintetiza date complexe și de a comunica rezultate științifice unui public academic.

## 🛠️ Metode utilizate
- Analiză Matematică și Calcul Fracționar.
- Analiză Complexă și Teoria Sistemelor.
- Metode Cantitative de Cercetare și Statistică.
